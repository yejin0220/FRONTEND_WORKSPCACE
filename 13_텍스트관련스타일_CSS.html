텍스트 관련 스타일(test~)

color : 텍스트의 색상을 지정

        선택자 {
            color : 색상명/ 16진수/rgv, rgba, hsl.....
        }

text-decoration: 텍스트에 줄을 긋거나 줄을 없앨 때 사용

        선택자{
            text-decoration : none/underline/overline/line-through;
        }
  • underline : 텍스트 영역 아래에 줄 긋기
  • line-through : 텍스트 영역 중간에 줄 긋기
  • overline : 텍스트 영역 위에 줄 긋기
네이버로 이동

text-transform : 영문 텍스트의 대소문자 변환 시 사용

         선택자{
            text-transform:uppercase/lowercase/capitalize;
         }
  • uppercase : 모든 영문자를 대문자로 변경해준다
  • lowercase : 모든 영문자를 소문자로 변경해준다
  • capitalize : 영문자의 첫 글자만 대문자로 변경해준다

text-align : 텍스트를 정렬할 때 사용

        선택자{
            text-align : left(기본값)/justify(양쪽정렬)/right/center;
        }

양쪽정렬

설레는 목숨이 열락의 사랑의 피고 인생을 아니다. 피에 피부가 별과 내는 되는 사막이다. 위하여 물방아 전인 싹이 같이, 그와 힘차게 힘차게 황금시대를 있는가? 이상은 이는 얼마나 이 용기가 풀이 얼음이 청춘의 청춘이 칼이다. 이상의 곧 위하여서 교향악이다. 전인 예수는 그러므로 우리 풍부하게 수 오아이스도 그리하였는가? 이상은 고동을 동력은 대고, 것은 것이다.보라, 쓸쓸하랴? 옷을 사라지지 굳세게 위하여서, 방황하였으며, 위하여서. 청춘은 어디 영원히 것은 것이다. 얼음이 청춘 우리의 이상 따뜻한 있는 것은 것이다. 쓸쓸한 하는 그들에게 관현악이며, 청춘 힘차게 것이다. 못하다 방황하여도, 불어 가지에 투명하되 봄바람이다. 공자는 끓는 못하다 옷을 이것이야말로 현저하게 그들은 밝은 부패뿐이다. 인류의 위하여 하는 끓는 얼음이 있는가? 예수는 목숨을 아름답고 바이며, 것이 부패뿐이다. 이는 인생을 이상의 풍부하게 봄바람을 이성은 위하여서, 실로 것이다. 얼마나 되려니와, 많이 그러므로 사막이다. 인생에 이것은 기관과 하였으며, 같이, 피다. 두손을 따뜻한 이상의 것이다.보라, 우리 과실이 그들은 사막이다. 웅대한 우리의 전인 같이, 청춘은 품에 만물은 운다. 같이 위하여서, 가치를 생의 별과 천하를 황금시대다. 꽃이 아니더면, 있는 천고에 열락의 가슴이 사는가 행복스럽고 열매를 것이다. 불러 구하지 투명하되 유소년에게서 운다. 인간에 희망의 고동을 보라. 같이 품으며, 천하를 봄바람이다. 커다란 인간은 이는 운다. 인간은 이것을 모래뿐일 있는 커다란 이것이다. 피는 두기 있음으로써 있을 길지 커다란 풀이 그리하였는가? 하였으며, 생명을 영락과 갑 희망의 운다. 용감하고 그와 온갖 꽃이 그들은 넣는 쓸쓸하랴? 긴지라 살 청춘의 구할 것이다.

오른쪽 정렬

구하지 안고, 꽃이 찾아 봄바람이다. 그들은 목숨을 같은 있으랴? 시들어 구하지 만천하의 불어 보내는 되려니와, 어디 뜨고, 인생에 봄바람이다. 어디 원질이 풀밭에 있으며, 약동하다. 열매를 그들은 이는 낙원을 이 이상의 밥을 수 인간에 듣는다. 가는 것이다.보라, 전인 뜨고, 황금시대다. 있을 군영과 많이 피고 장식하는 살았으며, 곳으로 풀밭에 봄바람이다. 소담스러운 얼음과 가슴에 피다. 피가 옷을 가진 이것이다. 보이는 평화스러운 그들은 이것이다. 얼마나 어디 가치를 것이다. 찾아다녀도, 구하기 바로 하는 이는 청춘의 인생에 청춘 것은 것이다. 행복스럽고 속에서 고동을 얼음에 밝은 이상을 것이다. 위하여서, 소금이라 무엇이 청춘의 그들은 속에 약동하다. 우는 있음으로써 석가는 없는 구하기 따뜻한 청춘에서만 사막이다. 같이 청춘의 같은 뜨고, 풀이 것이다. 피고, 거선의 주며, 아니다. 붙잡아 그들의 자신과 같지 천고에 아니더면, 이 얼마나 피가 있다. 구하기 넣는 때까지 그러므로 있으랴?

가운데 정렬

굳세게 곳으로 불어 있는 어디 피에 것은 노년에게서 쓸쓸하랴? 구하지 것이 앞이 때에, 그리하였는가? 무엇을 못할 천하를 길을 실현에 있는 가는 힘있다. 같은 고동을 것은 사는가 천하를 이상은 그와 것이다. 능히 밝은 보는 뼈 긴지라 아니다. 능히 못할 이성은 커다란 싶이 수 이것이다. 위하여서 찾아다녀도, 구하지 봄바람을 하여도 황금시대다. 맺어, 크고 목숨을 일월과 말이다. 얼마나 가치를 어디 끓는 많이 자신과 곧 투명하되 쓸쓸하랴?

text-shadow : 텍스트에 그림자 효과를 줄 때 사용

        선택자 {
            text-shadow : 가로거리(x) 세로거리(y) 번짐정도 색상;
        }
        
        -번짐정도와 색상은 생략 가능
        -가로거리와 세로거리가 양수일 경우 오른쪽 아래에 그림자 생성
        -가로거리와 세로거리가 음수일 경우 왼쪽 위에 그림자 생성
        -번짐정도가 0일 경우 그림자 색이 진하다
        -색상은 기본값이 글자색과 같다
