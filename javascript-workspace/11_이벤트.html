1. 이벤트 모델 종류

고전 이벤트 모델(기본 이벤트 모델)

요소 객체를 가지고 와서 해당 요소객체에 이벤트 속성에 접근한 후 이벤트 핸들러를 연결하는 방식
이벤트를 제거할 수도 있음.


인라인 이벤트 모델

요소 내부에 직접적으로 이벤트 속성을 제시해서 실행할 내용을 정의하는 방식
주로 script 태그 내에 정의되어 있는 함수를 호출하는 방식을 선호


표준 이벤트 모델(addEventListener)


2. 현재 이벤트가 발생한 해당 요소객체에 접근하는 방법


3. 태그별 기본적으로 가지고 있는 이벤트 제거

기본적으로 이벤트를 가지고 있는 요소
-a태그 : 클릭시 href에 제시되어 있는 url을 요청하는 기본이벤트를 가지고 있음
-form태그 -> 태그 내의 submit버튼 : 클릭시 사용자가 입력한 정보들을 action에 제시되어 있느 url로 제출하면서 요청하는 기본이벤트를 가지고 있음

*태그 내에 기본적으로 설정되어 있는 이벤트를 제고하고자 한다면, 이벤트 핸들러의 return값을 false로 리턴하면 기본이벤트는 무시가됨
