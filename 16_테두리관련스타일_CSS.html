테두리 관련 스타일(border~~)

        선택자{
            *테두리 스타일
            border-위치-style:none/dotted/dashed/solid/double/groove(패인 액자)/ridge(튀어나온 액자)/inset/outset;
            
            *테두리 두께
            border-위치-width : 테두리두께;
            
            *테두리 색상
            border-위치-color : 테두리색상;
            
            **위치 생략가능, 생략시 상하좌우 모두

            *박스에 그림자 효과
            box-shadow: 가로 세로 흐림 번짐 색상 ;

            *테두리에 모서리를 둥글게 하는 스타일
            border-위치-radius : 숫자;

            *****테두리의 두께, 스타일, 색상등을 한 번에 지정
            border-위치 : 두께 스타일 색상;
        }
