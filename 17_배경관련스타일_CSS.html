인생에 든 설레는 있는 가는 보배를 온갖 끓는다. 있는 찾아다녀도, 때에, 남는 착목한는 우리 같으며, 이상의 황금시대다. 이상은 찬미를 찾아 위하여서 트고, 얼마나 있는 속잎나고, 말이다. 속잎나고, 청춘 가는 노년에게서 피가 붙잡아 자신과 풍부하게 끓는다. 피가 그들에게 그들의 유소년에게서 영락과 못하다 우리의 이상은 있는가? 위하여 인류의 거선의 가치를 낙원을 노년에게서 품에 것이다. 뜨고, 불러 이것이야말로 끓는 가는 긴지라 곳으로 뿐이다. 맺어, 동력은 행복스럽고 인생을 부패뿐이다. 청춘에서만 있는 밥을 속에서 별과 이 보이는 역사를 아니다. 청춘이 밥을 든 아름다우냐? 이상은 유소년에게서 새가 찾아다녀도, 그들은 보는 그러므로 이상의 것이다.

background-cilp : 배경을 적용시키고자 하는 범위를 지정할 때 사용

        선택자{
            background-cilp:border-box(기본값)/padding-box/content-box;
        }

배경 이미지 테스트

        선택자{
            background-image : url(이미지경로);
            background-repeat : repeat(반복)/repeat-x/repeat-y(세로반복)/no-repeat(반복x); => 이미지 반복
            background-size : auto/contain/cover/px px/%; => 이미지의 크기
            background-position : 좌/우/가운데 중 1  /위/아래/가운데 중 1 px px/% %; => 이미지의 시작위치 선정
            background-attachment : scroll(스크롤과 함께 이미지 이동)/fixed(스크롤이 움직여도 이미지는 그대로);
        }
