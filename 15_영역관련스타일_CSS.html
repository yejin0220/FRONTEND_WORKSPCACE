기준 div

기준 div

padding

내용물 영역(content)과 테두리 영역(border)사이의 거리(여백)를 조절하는 스타일 속성 상하좌우로 지정한 px만큼 공백이 생기는 속성

컨텐츠 영역
padding:100px;

컨텐츠 영역
padding-top:50px; padding-left:100px;

컨텐츠 영역
padding : 50px 20px 10px 0px

margin

타 요소들간의 간격을 조정할 때 사용 margin auto : 화면의 크기가 변할때마다 좌우 간격을 자동으로 가운데로 맞춰 줌

컨텐츠 영역
margin:50px

컨텐츠 영역
margin-top:50px; margin-left:20px;

컨텐츠 영역
margin : 50px 30px 20px 0px
