이벤트 핸들러(이벤트 발생시 실행될 function) 연결 방법 종류

방법1. 이벤트 메소드를 통한 연결

        $("선택자").이벤트메소드(function(){
            해당 요소에 이벤트 발생시 실행할 내용;
        })

클릭클릭



방법2. on메소드를 이용하는 방법

        $("선택자").on("이벤트명",function(){
            해당 요소에 해당 이벤트 발생시 실행할 내용
        });

마우스 클릭 및 올려보세요.

        $("상위요소선택자").on("이벤트명","하위요소선택자", function(){

        });

키보드 관련 이벤트

- keydown / keypress : 키가 눌려질때 발생하는 이벤트
keydown : 키보드의 모든 키가 눌려질 때 발생
keypress : 키보드의 펑션키, 기능키, 한글키를 제외한 나머지가 눌려질 때 발생함
-keyup : 키보드가 떼어질때 발생하는 이벤트
