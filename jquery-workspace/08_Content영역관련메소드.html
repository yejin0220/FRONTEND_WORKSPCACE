Content영역(innerHTML, innerText)관련 메소드

* html("문구") 메소드 : innerHTML속성과 관련된 메소드

선택된 요소의 content영역(시작과 종료태그 사이)을 리턴해주거나 또는 변경해주는 메소드
-$("선택자").html() : 선택된 요소의 content영역의 값을 반환
-$("선택자").html("문구") : 선택된 요소의 content영역의 값을 문구로 변경해줌
전달하는 문구에 html태그가 있을 경우 태그로써 해석이 된다.

* text("문구") 메소드 : innerText 속성과 관련된 메소드

선택된 요소의 content 영역의 텍스트만 리턴해주거나 변경해주는 메소드
-$("선택자").text() : 선택된 요소의 content영역을 반환( 단, html 태그는 무시되고 텍스트만 전달됨/문자열형태로)
-$("선택자").text("문구") : 선택된 요소의 content영역을 전달된 값으로 변경
