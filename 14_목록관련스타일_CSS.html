목록 관련스타일(list~~~)

list-style-type : 블릿기호를 변겨시켜줄 때 사용

        *순서 없는 목록 태그(ul)
        선택자{
            list-style-type : disc(기본값)/circle/square/none(가장많이사용);
        }

        *순서 있는 목록 태그(ol)
        선택자{
            list-style-type:decimal/decimal-leading-zero(01,02~~/10보다 작을때만 숫자앞에 0붙여줌);
            list-style-type:lower-alpha(소문자)/upper-alpha(대문자);
            list-style-type:lower-roman(소문자로마)/upper-roman(대문자로마);
        }

순서 없는 목록 태그

  • HTML
  • CSS
  • JavaScript
  • jQuery

순서 있는 목록 태그

  1. HTML
  2. CSS
  3. JavaScript
  4. jQuery

list-style-position : 블릿기호의 위치를 조정할 때 사용

list-style-image : 블릿기호 대신에 이미지를 적용할 때 사용

        선택자{
            list-style-position : inside(안쪽으로 위치 움직임)/outside(기본값);
            list-style-type : url(적용시키고자 하는 이미지의 경로);
        }
