justify-content : 요소들을 가로로 정렬하도록 도와주는 속성

        선택자{
            justify-content :  flex-start/flex-end/center/sapce-between/space-around/space-evenly
            -flex-start : 요소들을 왼쪽으로 정렬(기본값)
            -flex-end : 요소들을 오른 쪽으로 정렬
            -center : 가운데 정렬
            -space-between : 요소들 사이에 동일한 간격을 부여하여 정렬
            -space-around : 요소들 주위(양옆)에 동일한 간격을 부여하여 정렬
            -space-evenly : 요소들 마다 동일한 간격을 부여하여 정렬
        }

FLEX

flex-direction : flex요소들의 정렬방향 지정 가능(행, 열 단위)

 
        선택자 {
            flex-direction : row/row-reverse/column/column-reverse
            -row : 요소들을 텍스트 방향과 동일하게 정렬(왼쪽 -> 오른쪽)
            -row-reverse : 요소들을 텍스트의 반대방향으로 정렬(오른쪽 -> 왼쪽)
            -column : 요소들을 위에서 아래로 정렬(블럭처럼)
            -column-reverse : 요소들을 아래에서 위로 정렬
        }

align-items : 요소들을 세로 기준으로 정렬하게 도와주는 속성

        선택자{
            align-items : flex-start/flex-end/center/stretch
            -flex-start : 요소들을 맨 위로 정렬(맨 왼쪽)
            -flex-end : 요소들을 맨 아래로 정렬(맨 오른쪽)
            -center : 가운데로 정렬
            -stretch : 요소들을 컨테이너(부모요소)의 크기에 맞도록 늘려주는 속성(기본값)
        }

        *-column : 요소들을 위에서 아래로 정렬(블럭처럼) 로 바꿔준 후에 속성 적용해야함(복합적으로 사용하려면)

order : 특정 요소만 정렬 순서를 바꿔주는 속성(전체)

flex-wrap : 요소들을 한 줄로 정렬할지, 여러줄로 정렬할지 정하는 속성

        선택자{
            flex-wrap : nowrap/wrap/wrap-reverse;
            -nowrap : 요소들을 한줄로만 정렬시키는 속성
            -wrap : 요소들을 여러줄로 정렬시키는 속성
            -wrap-reverse : 요소들을 여러줄로 반대로 정렬시키는 속성
            *화면의 크기를 줄이고 늘릴때마다 자동으로 정렬시켜줌
        }
